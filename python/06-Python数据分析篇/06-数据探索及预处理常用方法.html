数据集的合并¶

根据两组数据中的关键字key来合并

使用条件：将两个表中有相同的列名

In [98]:
#merge
import pandas as pd

df1 = pd.DataFrame({'key':['a','b','c'],'data1': range(3)})
df1
Out[98]:

1 b 1
2 c 2

df2 = pd.DataFrame({'key':['a','b','d'],'data2':range(3)})
df2
Out[99]:
key data2
0 a 0
1 b 1
2 d 2

Out[100]:

concat：轴向连接

不管列名，直接加到一起，可以加到后面、也可以加到右边

  • axis=0为加到后面
  • axis=1为加到右边
