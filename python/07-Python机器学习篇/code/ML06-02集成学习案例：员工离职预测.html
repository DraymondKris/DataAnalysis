字段说明：

  • satisfaction_level:对公司的满意度
  • last_evaluation：最近一次考核
  • number_project：项目数
  • average_montly_hours：平均每月工作时长/小时
  • time_spend_company：工作年限
  • Work_accident：是否有过工作事故
  • left：是否离职
  • promotion_last_5years：过去5年是否晋升
  • sales：岗位类别
  • salary：薪资水平，高、中、低
